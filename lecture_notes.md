# Section 16: Django Level One - Basics

## Check Django version
```bash
python -m django --version
```

## Add Django project

```bash
cd src
django-admin startproject [project_name]
```

## Create app
```bash
cd [project_dir]
python manage.py startapp [app_name]
```

## Start project

```bash
cd [project_dir]
python manage.py runserver
```

## Run migration

```bash
python manage.py migrate
```
